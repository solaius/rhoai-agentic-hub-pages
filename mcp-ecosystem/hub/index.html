What's New

  • MCP Gateway v0.7.1 released (June 18, 2026) — Major performance gains: p95 latency at 500 servers dropped from 390ms to 47ms. Includes TLS hairpin fix and per-server SNI fix. Note: following the official MCP Go SDK migration (July 2026), some post-migration performance regressions are under investigation; re-validation in progress.
  • MCP Lifecycle Operator v0.2.0 released (June 17, 2026) — Improved stability and multi-namespace support. Bundled within RHOAI operator releases.
  • RHOAI 3.5 targeting August 2026 — MCP Registry DP, Gateway integration TP, MCPLO promotion to TP. Full ecosystem DevPreview milestone.
  • End-to-end implementation guide published — Fernando Lozano's practical notes fill MCP deployment gaps: minimal path (playground without gateway) and full production setup.
  • RHAISTRAT-1339 AI Hub AI Asset Delivery approved — 28 RFEs approved for MCP Registry, Catalog, Gateway, Lifecycle Operator, Agent Registry, and Skills Registry.
