Strategic Outcome

RHAISTRAT-1339: AI Hub AI Asset Delivery for Agentic Solutions

Umbrella outcome for the full MCP Ecosystem delivery.

The roadmap above represents the unified delivery of all components required to achieve the strategic outcome: a governed, federated AI asset ecosystem for agentic solutions on Red Hat OpenShift AI.

Key dependencies:

  • Catalog depends on Lifecycle Operator for deployment primitive
  • Gateway depends on RHCL for infrastructure
  • Registry depends on Catalog + Gateway + MCPLO for integration points
  • Gen AI Studio depends on Registry (ConfigMap bridge in 3.4; registry-backed once Registry TP lands in 3.6 EA1)

GA readiness criteria: All four components reach GA, registry→gateway auto-config works, end-to-end flow (Catalog → Deploy → Govern → Enforce → Consume) validated in production.
