STANDARDS WORKSTREAM

Build Toward Standards from Day One

There is no memory standard. That absence is an opportunity. The standards workstream runs in parallel with product work — Phase 0’s primary deliverable.

Phase 0 Deliverable

MCP Memory Convention SEP

Standards Enhancement Proposal formalizing the de facto memory-tool naming convention: canonical tool set, minimal record format, scope model. The 3.7 substrate’s MCP surface tracks it.

Phase 0 Deliverable

A2A AgentCard Memory Binding

Optional extension declaring an agent’s memory services — transport, scope model, access method. Makes memory discoverable and governable via AgentCards.

Longer Horizon

AAIF Memory Project

Neutral governance home for a memory schema, API contract, and portability spec. The MCP/AGENTS.md donation model. Socialize from Phase 0; formal proposal post-3.7.

“A platform that ships memory primitives before standards converge gets to shape them. A platform that waits will inherit them.”

Four structural differentiators that competitors cannot easily replicate.

RISKS & DEPENDENCIES

Eyes Open

The strategic risk is timing, not capability. The hyperscalers and Oracle ship today.

Proprietary-API Lock-In

A bespoke memory API built before standards converge inherits migration cost. Mitigated: standards workstream from day one.

Audit Trail Under-Delivered

EU AI Act August 2026; no internal candidate ships a working audit trail. Mitigated: GA gate held firm; minimum write-event log at Dev Preview.

Platform Transition Parity

The gateway-native architecture may not reach full primitive parity by Phase 1. Mitigated: substrate-agnostic API surface allows transitional fallback.

Market Consolidation

$62M+ into 5+ startups with near-identical approaches. Mitigated: partner products only as pluggable backends, never the substrate.

Org Absorption

Competing priorities; ~3-month release cadence. Mitigated: Phase 0 is near-zero cost; Phase 1 is Dev Preview; sizing is explicit per phase.

Phase 2 Scope Expansion

Adversarial defense and benchmarking add new GA criteria. Mitigated: size early; adversarial is security-critical, benchmarking can be phased.
