v0.7.0 Performance Gains

8× Latency Reduction Under Load

p95 latency at 500 MCP servers dropped from 390ms to 47ms.

8-hour soak test results:

  • 44.3M requests processed
  • 1,540 RPS sustained throughput
  • Zero broker restarts
  • Zero memory leaks

This performance overhaul makes the gateway production-ready for high-scale enterprise deployments.

This result predates the July 2026 migration to the official MCP Go SDK. A partner performance-testing team's post-migration run found HTTP 500s under load at high concurrency and broker connection timeouts at high server counts; re-validation of these performance figures against the new SDK is in progress (see the SDK Migration open question).

Upstream MCP Spec

MCP Spec Release Candidate: Finalizes July 28, 2026

The Release Candidate has been live since May 21, 2026 and finalizes July 28, 2026. It carries several breaking changes: the streamable HTTP transport goes stateless (no initialize handshake, no Mcp-Session-Id), server/discover becomes mandatory, SSE resumability is removed, OAuth Dynamic Client Registration is deprecated in favor of Client ID Metadata Documents, and tool schemas move to full JSON Schema 2020-12.

MCP Gateway is the most impacted RHOAI component: the GA-roadmap "Session Management (Resumable Sessions)" feature needs complete redefinition, since protocol-level sessions no longer exist.

Status: RC in its validation window; Gateway team assessing rework scope ahead of the July 28 finalization.

See the full MCP Spec RC Impact Analysis for component-by-component detail.
