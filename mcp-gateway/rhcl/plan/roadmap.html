v0.7.0 Performance Gains

8× Latency Reduction Under Load

p95 latency at 500 MCP servers dropped from 390ms to 47ms.

8-hour soak test results:

  • 44.3M requests processed
  • 1,540 RPS sustained throughput
  • Zero broker restarts
  • Zero memory leaks

This performance overhaul makes the gateway production-ready for high-scale enterprise deployments.
