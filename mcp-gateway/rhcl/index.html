What's New

  • v0.7.1 released (June 18, 2026) — Major performance gains: p95 latency at 500 servers dropped from 390ms to 47ms. Includes TLS hairpin fix and per-server SNI fix. This result predates the July 2026 migration to the official MCP Go SDK: a partner performance-testing team observed significant post-migration regressions (HTTP 500s under load, broker connection timeouts at high server counts), and re-validation is in progress.
  • RHCL 1.4 GA (June 2026) — MCP Gateway ships as Technology Preview with OCP 4.22 support.
  • MCP spec RC finalizes July 28, 2026: Release Candidate live since May 21, 2026, with several breaking changes: streamable HTTP goes stateless (no initialize, no Mcp-Session-Id), mandatory server/discover, SSE resumability removed, OAuth DCR deprecated for Client ID Metadata Documents, and JSON Schema 2020-12 in tool schemas. MCP Gateway is the most impacted component: see the full impact analysis.
  • Active customer POC traction — evaluations underway with a large enterprise OEM customer, a major telecommunications provider, and a strategic hardware partner's joint-solution POC.
